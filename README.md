# Expense Tracker (Трекер расходов)

**Автор:** [Эверт Глеб]

## Описание программы

Expense Tracker - это десктопное приложение с графическим интерфейсом для отслеживания личных расходов. Программа позволяет вести учет финансов, категоризировать расходы и анализировать траты за различные временные периоды.

### Основные возможности:
- Добавление расходов с указанием суммы, категории и даты
- Автоматический подсчет общей суммы расходов за выбранный период
- Фильтрация расходов по категориям
- Фильтрация по временному периоду
- Сохранение данных в JSON формате
- Возможность удаления отдельных записей
- Экспорт и импорт данных

## Технологии

- **Python 3.7+**
- **PyQt5** - GUI фреймворк
- **JSON** - формат хранения данных
- **Git** - система контроля версий
